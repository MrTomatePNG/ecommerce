Bem-vindo à API E-commerce

Versão 1.0

Endpoints Disponíveis:

  • Produtos:
    • GET /produtos
    • POST /produtos
    • GET /produtos/{id}
    • PUT /produtos/{id}
    • DELETE /produtos/{id}
  • Usuários:
    • POST /usuarios
    • GET /usuarios/{id}
    • PUT /usuarios/{id}
    • DELETE /usuarios/{id}
  • Pedidos:
    • POST /pedidos
    • GET /pedidos/{id}
    • GET /pedidos/usuario/{usuarioId}
    • PATCH /pedidos/{id}/status
